# 数据文件格式说明

## 总体要求
- 上传一个 `xlsx` 文件。
- 文件中至少包含两个 sheet。
- 通过页面中的 `Sheet A` 与 `Sheet B` 选择要参与绘图的两个 sheet。

## 通路对比（对应 `scripts/通路对比.py`）
每个 sheet 需要至少两列：

| 列序号 | 含义 | 示例 |
|---|---|---|
| 第1列 | 通路名称（pathway） | `WNT` |
| 第2列 | 概率或信息流数值（probability） | `0.024` |

说明：
- 程序会忽略 `total/sum/总计/总和/合计` 这类汇总行。
- 两个 sheet 会按通路名称自动对齐，绘制相对占比堆叠图。

## 相关性热图（对应 `scripts/相关性热图.py`）
每个 sheet 格式：

| 列序号 | 含义 | 示例 |
|---|---|---|
| 第1列 | 基因 ID / 名称 | `Runx2` |
| 第2列起 | 数值表达矩阵（样本或条件） | `1.23`, `0.98`, ... |

说明：
- `Sheet A` 与 `Sheet B` 会按第一列基因名求交集后进行 Pearson 相关计算。
- 支持三种相关模式：全部、仅正相关、仅负相关。
