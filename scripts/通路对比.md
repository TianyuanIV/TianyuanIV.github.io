# 通路对比（通路相对信息流）

## 输入文件要求

- 文件类型：`xlsx`
- 使用一个 Excel 文件，选择其中两个 Sheet 进行对比
- 每个 Sheet 至少包含前两列：
  - 第 1 列：通路名称（例如 `WNT`, `TGF-beta`）
  - 第 2 列：数值（概率/信息流强度等，需为数值）

## 数据示例

| Pathway | Probability |
| --- | ---: |
| WNT | 0.23 |
| TGF-beta | 0.11 |
| Notch | 0.34 |

## 说明

- 系统会自动忽略汇总行（如 `Total`、`Sum`、`总计`、`合计` 等）。
- 绘图结果为堆叠条形图，展示每个通路在两个 Sheet 中的相对占比。
